
# Flanny App Home Button

This code can be included on any Flanny.app to automatically add a home button.

## Installation

Add the following script tag to every page you want the home button to appear.

```html
<script src="https://flanny.app/flanny-app-home-button/add-home-button.js"></script>
```

Modify the home button by calling `configureHomeButton`

```html
<script>
  configureHomeButton({
    backgroundColor: '#FF5733'
  });
</script>
```
